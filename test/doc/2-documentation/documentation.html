GET /:grid5000-resource/versions[.:format]

Get the list of all versions of a particular resource.

URI parameters
grid5000-resource:the URI of the grid5000 resource.
format:the requested format [json,xml] if you do not use the Accept header in your HTTP request.
Query parameters
limit:maximum number of versions to return.
Content-Types
application/json:JSON
application/xml:XML
Status codes
200:OK, the response contains the list of the versions of the requested grid5000 resource.
404:the requested grid5000 resource cannot be found.
406:the requested format is not available.
Usage

Get the 2 latest versions of the Rennes site: 


    GET /grid5000/sites/rennes/versions?limit=2 HTTP/1.1
    Accept: application/json

GET /:grid5000-resource/versions/:version[.:format]

Get info about a specific version.

URI parameters
grid5000-resource:the URI of the grid5000 resource.
version:the version id (40 characters long) or a UNIX timestamp.
Content-Types
application/json:JSON
application/xml:XML
Status codes
200:OK.
404:the requested grid5000 resource cannot be found, or the requested version does not exist.
406:the requested format is not available.
GET /:grid5000-resource[.:format]

Get a specific version of a Grid5000 resource.

URI parameters
grid5000-resource:the URI of the grid5000 resource.
Query parameters
version:the requested version. It can be a version id (40 characters), or a UNIX timestamp [default=empty (most recent version is used)].
depth:the number of nested sub-resources to resolve [default=1].
resolve:a list of comma separated sub-resources names to resolve (to be used with the depth parameter) [default=all].
Content-Types
application/json:JSON
application/xml:XML
application/zip:the ZIP format will return a zip archive containing the set of directories and files corresponding to the required data, with all its sub-resources.
Status codes
200:OK, the response contains the description of the resource as it was at the requested version.
404:the requested grid5000 resource cannot be found, or the requested version does not exist.
406:Returns 406 if the requested format is not available.
